UI/UX DESIGN

UI/UX design requires a deliberate analysis of factors influencing the interaction between the user and the product. To do this I consider the following equations:

User Experience = Why + What + How


  • Why involves the users’ motivation for adopting a product, whether the the user relates to the task they wish to perform with the product, or the values and views associated with the ownership and use of the product.
  • What addresses the tasks people can complete with the product.
  • How relates to the design of functionality in an accessible and aesthetically pleasant way.


User Interface = Visual + Interactions


  • Visual involves guiding the user through a product’s interface via interactive elements across all sizes/platforms.
  • Interactions are visual responses when a user communicates with a product.
